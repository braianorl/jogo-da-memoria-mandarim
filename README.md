# Jogo da Memória em Mandarim

Este é um jogo da memória desenvolvido com HTML, CSS e JavaScript. O objetivo é combinar pares de cartas com caracteres em mandarim.

## Como jogar
- Abra o arquivo `index.html` em um navegador.
- Clique nas cartas para revelar os caracteres e encontrar os pares.

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Como executar
1. Clone o repositório: `git clone https://github.com/braianorl/jogo-da-memoria-mandarim.git`
2. Abra o arquivo `index.html` no navegador.
